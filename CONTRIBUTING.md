#  Contributing to EcoLoop

Thank you for your interest in contributing to EcoLoop. We welcome contributors of all skill levels. Please follow the guidelines below to ensure a smooth collaboration process.

---

##  How to Contribute

1. Fork the repository
2. Clone your fork locally
3. Create a new branch

   * Use a meaningful name (e.g., `feature/add-navbar`, `fix/readme-typo`)
4. Make your changes
5. Commit your changes with a clear message
6. Push to your fork
7. Open a Pull Request (PR)

---

##  Contribution Guidelines

* Follow the existing project structure and coding style
* Write clean, readable, and maintainable code
* Keep pull requests small and focused
* Add comments where necessary
* Ensure your changes do not break existing functionality

---

##  Commit Message Guidelines

Use clear and descriptive commit messages:

Examples:

* `feat: add product card component`
* `fix: resolve navbar responsiveness issue`
* `docs: update README with setup instructions`

---

##  Before Submitting a PR

* Ensure the project runs without errors
* Test your changes locally
* Check for formatting and linting issues
* Make sure your PR addresses the issue clearly

---

##  Issue Labels

We use the following labels to organize issues:

* **good first issue** – Suitable for beginners
* **bug** – Something is not working correctly
* **enhancement** – New feature or improvement
* **documentation** – Improvements or additions to documentation
* **help wanted** – Extra attention or help is needed
* **question** – Further clarification is required
* **duplicate** – Issue already exists
* **invalid** – Not a valid issue or request
* **wontfix** – This will not be worked on

---

## 💬 Communication

* Use GitHub Issues for reporting bugs or requesting features
* Use Discussions (if enabled) for questions and ideas
* Be respectful and constructive in all interactions

---

##  Code of Conduct

Please follow the Code of Conduct to maintain a welcoming and inclusive environment for everyone.

---

We appreciate your time and effort in contributing to EcoLoop.
